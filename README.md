# ms_toollib

扫雷算法工具箱。已发布到：

- crates.io
- pypi.org
- npmjs.com

可在Rust、Python/Windows、Nodejs、VUE3/Typescript等平台使用，并提供案例，可快速上手。

Algorithms for minesweeper, published on various platforms.

- 目前主要文档见[https://docs.rs/ms_toollib](https://docs.rs/ms_toollib)。
